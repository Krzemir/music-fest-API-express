<h3>Music Festival App with seats reservation system</h3>

<p>An application for a music festival that allows you to book tickets online in real time using Websocket technology.
An educational project focused on back-end learning.</p>

Deployed on: https://music-fest-api-express.krzemir.repl.co/
<p>(very limited server hosting performance, please be patient :) )</p>


<h4>Stack:</h4>
<p>express, API, MongoDB, Mongoose, Websocket, Mocha tests API</p>
